Description

This Breathing System allows asthma patients and doctors registering for the system. Patients can periodically self-report their breathing rate. Doctors can login to the system to view patients' data. The following functions are to be provided as an Android application:

  1. permit patients and doctors to register for the asthma health diary service
  2. permit patients to record their breathing data: number of breaths taken in a 1 minute interval
  3. prompt patients to perform a daily breathing test and report the data
  4. store patient data in an sqlite database on the patient's Android phone
  5. permit the patient view their breathing data in a graphical
  6. sync patient breathing data with database and web service
  7. permit doctors to browse, fetch, and display, historical data for their patients
  8. provide a stop watch to help patients capture their breathing data
